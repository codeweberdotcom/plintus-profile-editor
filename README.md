# Plintus Profile Editor

Редактор профилей плинтусов по сетке с возможностью рисования линий, скруглений и редактирования.

## Функциональность

- ✅ Рисование линий под углом 90 градусов
- ✅ Отображение длины линий
- ✅ Создание скруглений (дуг)
- ✅ Сетка с привязкой точек
- ✅ Редактирование элементов
- ✅ Удаление элементов
- ✅ Сохранение/загрузка профилей

## Установка

1. Скопируйте папку плагина в `wp-content/plugins/`
2. Активируйте плагин через админ-панель WordPress
3. Установите зависимости: `npm install`
4. Соберите React приложение: `npm run build`

## Разработка

### Установка зависимостей
```bash
npm install
```

### Режим разработки (с автообновлением)
```bash
npm run dev
```

### Production сборка
```bash
npm run build
```

## Использование

1. Перейдите в меню "Plintus Profiles" в админ-панели
2. Создайте новый профиль или откройте существующий
3. Используйте инструменты для рисования:
   - **Select** - выбор элементов
   - **Line** - рисование линий (только под 90°)
   - **Arc** - создание скруглений
   - **Delete** - удаление элементов
4. Настройте сетку в панели инструментов
5. Редактируйте свойства элементов в боковой панели
